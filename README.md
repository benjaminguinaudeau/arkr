# arkr

<!-- badges: start -->
<!-- badges: end -->

The goal of arkr is to obtain data on the structure of the ETF proposed
by ark.

## Installation

``` r
remotes::install_github("ropensci/tabulizer")
devtools::install_github("benjaminguinaudeau/arkr")
```

``` r
library(arkr)
```
